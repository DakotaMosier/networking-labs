# Networking Labs Portfolio

This repository contains hands-on networking labs performed using macOS Terminal to simulate real-world Help Desk and IT Support troubleshooting scenarios.

These labs demonstrate foundational networking skills including IP configuration, DHCP verification, DNS resolution, connectivity testing, and routing analysis.

---

## Objectives

The purpose of this project is to:

• Develop practical networking troubleshooting skills  
• Gain hands-on experience with core network commands  
• Build a technical portfolio for entry-level IT Help Desk positions  
• Demonstrate real-world diagnostic methodology  

---

## Skills Demonstrated

• IP address verification using ifconfig  
• DHCP troubleshooting and validation  
• Network connectivity testing using ping  
• DNS resolution using nslookup  
• Network path analysis using traceroute  
• Routing table analysis  

---

## Tools Used

• macOS Terminal  
• ifconfig  
• ipconfig  
• ping  
• nslookup  
• traceroute  
• netstat
